費用の最小化・利潤の最大化
Section 1独占と利潤

需要関数が１次関数のとき、独占企業の利潤がなぜ２次関数になるのかを学びます。

📖 問題設定：井戸水の独占販売

あなたは村で唯一の井戸を保有している。井戸水を１回汲んで販売するのに常に2の費用がかかり、固定費用は0であるとする。

需要関数

$$D(p) = 8 - p$$

いくらでいくつ販売すると利潤（総収入 − 総費用）が最大化されるだろうか？

🎛 利潤を最大化する生産量を探そう

逆需要関数は $P(q) = 8-q$ なので、価格は $p=8-q$ で決まります。生産量 $q$ を動かして、利潤がどう変化するか観察しましょう。

生産量 q = 3
価格 $p = 8-q$5
総収入 $R = pq$15
総費用 $C = 2q$6
利潤 $\pi = R-C$9
$q=3$ のとき利潤は最大値 9 をとります！
利潤 $\pi=6q-q^2$
現在の点
最大点
🧮 利潤関数を式で求める

逆需要関数 $P(q) = 8-q$ より、収入は

$$R = p \times q = (8-q)q = 8q - q^2$$

１単位の生産に追加的な費用が 2 かかるため、費用は $C = 2q$。よって利潤は

$$\pi = R - C = (8q-q^2) - 2q = 6q - q^2$$

利潤 $\pi$ が生産量 $q$ の関数になっていますが、今回は１次関数ではなく $q^2$ の項が登場しました。これが ２次関数 です。

定義：２次関数

説明変数の最大の次数が 2 である関数を２次関数という。一般に $y = ax^2+bx+c$ （$a\neq0$）の形で表される。

📝 確認問題
需要関数が $D(p)=10-p$、追加的な費用が常に 4 である独占企業の利潤関数 $\pi(q)$ はどれか？（逆需要関数は $P(q)=10-q$）
Section 2２次関数の最大値・最小値

放物線の形と頂点の関係から、最大値・最小値の在り処を理解しましょう。

$y=ax^2$ で $a>0$

原点を通る下に凸の谷型の放物線。
$x=0$ で最小値 $y=0$ をとる。

$y=ax^2$ で $a<0$

原点を通る上に凸の山型の放物線。
$x=0$ で最大値 $y=0$ をとる。

平行移動した形

$$y = a(x-s)^2 + t$$

放物線 $y=ax^2$ を $x$ 軸方向に $s$、$y$ 軸方向に $t$ だけ平行移動したもの。

  • $a>0$ ⇒ 下に凸、$x=s$ で最小値 $y=t$
  • $a<0$ ⇒ 上に凸、$x=s$ で最大値 $y=t$
🎛 グラフをいじって頂点を観察しよう
係数 a = 1
平行移動 s = 0
平行移動 t = 0
関数の式$y = x^2$
頂点（凸の向き）下に凸
最大値・最小値$x=0$ で最小値 $y=0$
$y=a(x-s)^2+t$
頂点 $(s,t)$
$a$ の符号で「上に凸・下に凸」が決まり、$(s,t)$ が頂点（最大値・最小値をとる点）の座標になります。
📝 確認問題
$y = -2(x-1)^2 + 5$ について正しいものはどれか？
Section 3平方完成で最大値・最小値を求める

一般の２次関数 $y=ax^2+bx+c$ を $y=a(x-s)^2+t$ の形に変形する手順です。

🔢 一般的な手順（平方完成）

$x^2$ の係数 $a$ でくくり、$\left(\dfrac{b}{2a}\right)^2$ を加えて引くことで平方の形を作ります。

出発点
$$y = ax^2+bx+c$$
ステップ１：$a$ でくくる
$$y = a\left(x^2 + \frac{b}{a}x\right) + c$$
ステップ２：$\left(\dfrac{b}{2a}\right)^2$ を加えて引く
$$y = a\left(x^2 + \frac{b}{a}x + \left(\frac{b}{2a}\right)^2 - \left(\frac{b}{2a}\right)^2\right) + c$$
ステップ３：平方の形にまとめる
$$y = a\left(x+\frac{b}{2a}\right)^2 - \frac{b^2}{4a} + c = a\left(x+\frac{b}{2a}\right)^2 + \frac{4ac-b^2}{4a}$$
結論 ✅

$x = -\dfrac{b}{2a}$ のとき、

$$a>0 \implies \text{最小値 } y=\frac{4ac-b^2}{4a}, \qquad a<0 \implies \text{最大値 } y=\frac{4ac-b^2}{4a}$$
🧮 井戸水の利潤関数に適用する

第1節で求めた $\pi = 6q - q^2 = -q^2+6q = -(q^2-6q)$ を平方完成してみましょう。

出発点：$q^2$ の係数でくくる
$$\pi = -(q^2 - 6q)$$
ステップ１：$-6$ の半分 $-3$ の２乗 $9$ を加えて引く
$$\pi = -(q^2 - 6q + 9 - 9)$$
ステップ２：平方の形にまとめる
$$\pi = -(q^2-6q+9) + 9 = -(q-3)^2 + 9$$
結論 ✅
$$q = 3 \ \text{で利潤が最大化し} \ \pi = 9$$

このとき価格は $p = 8-3 = 5$、収入 $R=15$、費用 $C=6$、利潤 $\pi = 15-6=9$ ✓

🎛 頂点を確かめよう

スライダーで $q$ を動かし、$\pi=-(q-3)^2+9$ の値が頂点 $(3,9)$ で最大になることを確認しましょう。

生産量 q = 3.0
$\pi(q) = -(q-3)^2+9$9.00
頂点（最大値）$(3,\ 9)$
頂点と一致しています。これが最大値です！

Section 4練習問題：ラーメン屋の価格設定

教科書 例題2.1 を、価格 $p$ の関数と生産量 $q$ の関数の２通りで解いてみましょう。

📖 問題文

あなたはラーメン屋チェーンのオーナーで、A市のショッピングセンターに新しくラーメン屋をオープンさせるか検討している。このセンターには他にラーメン屋はなく、需要は１日あたり

$$D(p) = 500 - \frac{1}{2}p \quad (0\le p \le 1000)$$

という需要関数で与えられる。ラーメン１杯を作る費用は人件費等を含めてちょうど400円である。

  1. 利潤を価格 $p$ の関数 $f(p)$、生産量 $q$ の関数 $g(q)$ として２通りの方法で求めなさい。
  2. 利潤を最大にするにはラーメンを１杯いくらで売ればよいか。
🔢 ２通りの解き方

価格 $p$ で売ると、需要量（販売数量）は $D(p)$ 杯。１杯あたりの利益は $(p-400)$ 円なので…

利潤関数を立てる
$$f(p) = (p-400)D(p) = (p-400)\left(500-\frac{1}{2}p\right)$$
ステップ１：展開する
$$f(p) = -\frac{1}{2}p^2 + 700p - 200000$$
ステップ２：平方完成する
$$f(p) = -\frac{1}{2}(p^2-1400p) - 200000 = -\frac{1}{2}(p-700)^2 + 45000$$
結論 ✅
$$p=700 \ \text{のとき利潤が最大化し} \ f(700)=45000 \ \text{円}$$

$0\le 700\le 1000$ なので定義域内。最適価格は 700円。

$D(p)=500-\frac12 p$ を $p$ について解くと、逆需要関数 $p = P(q) = 1000-2q$。生産量 $q$ あたりの利益は $(P(q)-400)$ 円なので…

利潤関数を立てる
$$g(q) = (P(q)-400)q = (1000-2q-400)q = (600-2q)q$$
ステップ１：展開する
$$g(q) = -2q^2 + 600q$$
ステップ２：平方完成する
$$g(q) = -2(q^2-300q) = -2(q-150)^2 + 45000$$
結論 ✅
$$q=150 \ \text{のとき利潤が最大化し} \ g(150)=45000 \ \text{円}$$

このとき価格は $p = P(150) = 1000-300 = 700$ 円となり、$f(p)$ の結果と一致します ✓

価格 $p$ で考えても、生産量 $q$ で考えても、同じ最適点（$p=700$ 円、$q=150$ 杯、利潤 $45{,}000$ 円）にたどり着きます。
🎛 価格を変えて利潤の変化を観察しよう
価格 p = 700 円
需要量 $q=D(p)$150
１杯あたりの利益300円
利潤 $f(p)$45,000円
$p=700$ 円のとき利潤は最大値 45,000円 です！
$f(p)=-\frac12(p-700)^2+45000$
現在の点
最大点

Section 5まとめ

第4回の要点を整理しましょう。

✅ 今回のポイント
1
独占企業の利潤と２次関数
需要関数が１次関数だと、独占した企業の収入（および利潤）が生産量の２次関数として表せる。
2
２次関数の最大値・最小値
$y=a(x-s)^2+t$ の頂点 $(s,t)$ で、$a>0$ なら最小値、$a<0$ なら最大値をとる。
3
平方完成
一般の２次関数 $y=ax^2+bx+c$ は $y=a\left(x+\dfrac{b}{2a}\right)^2+\dfrac{4ac-b^2}{4a}$ の形に変形でき、頂点が求まる。
4
微分との関係
２次関数の最大値・最小値は、平方完成だけでなく、後半で学ぶ「微分」によっても求めることができる。
