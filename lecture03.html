Section 1連立方程式

２つの条件を同時に満たす解を求める方法を学びます。

📖 問題設定

Aさんは明日の遠足に向けてお菓子を買います。

  • 飴：100円、チョコレート：200円
  • おやつ予算：ちょうど500円を全額使う
  • 鞄の都合：合計4個しか買えない

飴を $x$ 個、チョコを $y$ 個とすると…

連立方程式
$$\begin{cases} 100x + 200y = 500 & \text{（予算制約）}\\ x + y = 4 & \text{（個数制約）}\end{cases}$$
🔢 解き方

一方の変数を他方の式に代入して解く方法です。

初期状態
$$\begin{cases} 100x + 200y = 500 \cdots①\\ x + y = 4 \cdots② \end{cases}$$
ステップ１：②を $x$ について解く
$$x = 4 - y$$
ステップ２：①に代入する
$$100(4-y) + 200y = 500$$
ステップ３：展開・整理
$$400 - 100y + 200y = 500 \implies 100y = 100$$
ステップ４：解を求める ✅
$$y = 1 \quad \therefore \quad x = 4-1 = 3$$

確認：$100\times3+200\times1=500$ ✓、$3+1=4$ ✓

一方の式を定数倍して加減し、変数を消去する方法です。

初期状態
$$\begin{cases} 100x + 200y = 500 \cdots①\\ x + y = 4 \cdots② \end{cases}$$
ステップ１：②×100
$$100x + 100y = 400 \cdots②'$$
ステップ２：①－②' で $x$ を消去
$$(100x+200y)-(100x+100y) = 500-400 \implies 100y = 100$$
ステップ３：解を求める ✅
$$y=1 \quad \therefore \quad x = 4-1 = 3$$
🎛 インタラクティブ：パラメータを変えてみよう
予算 B = 500円
最大個数 N = 4個
飴の個数 x3
チョコの個数 y1
費用合計500円 ✓
予算 $B$ と最大個数 $N$ を変えると、直線の位置が変わり交点（解）が移動します。

Section 2需要関数・供給関数・均衡

市場での価格と数量がどのように決まるかを数式で表します。

定義：需要関数

価格 $p$ が決まると需要量 $D$ が決まる関数。
例：$D(p) = \alpha - \beta p$（右下がり）

定義：供給関数

価格 $p$ が決まると供給量 $S$ が決まる関数。
例：$S(p) = \gamma p$（右上がり）

定義：均衡価格・均衡取引量

$D(p^*) = S(p^*)$ を満たす価格 $p^*$ を均衡価格、取引量 $q^* = D(p^*)$ を均衡取引量という。

🧮 練習問題と解法ステップ

$D(p)=100-4p$、$S(p)=6p$ のとき、均衡価格と均衡取引量を求めよ。

均衡条件を立てる
$$D(p^*) = S(p^*) \implies 100 - 4p^* = 6p^*$$
ステップ１：$p^*$ について解く
$$100 = 10p^* \implies p^* = 10$$
ステップ２：均衡取引量を求める ✅
$$q^* = D(10) = 100-40 = 60 \quad \bigl(= S(10)=60 \text{ ✓}\bigr)$$

Section 3余剰分析

市場取引から生まれる「利益」をどう測るかを学びます。

定義：支払意思額（WTP）

買い手が財の1単位に対して支払う意思のある最大の金額。

定義：受入意思額（WTA）

売り手が財の1単位を売るために受け入れる最小の金額。

定義：消費者余剰（CS）

支払意思額 − 実際に払った価格。
価格より上、需要曲線より下の三角形面積。

$$CS = \frac{1}{2}(p_{max} - p^*)\cdot q^*$$
定義：生産者余剰（PS）

実際に受け取った価格 − 受入意思額。
価格より下、供給曲線より上の三角形面積。

$$PS = \frac{1}{2}p^* \cdot q^*$$
🎛 余剰を可視化する

需要：$D(p)=1000-p$、供給：$S(p)=3p$ → 均衡：$p^*=250$、$q^*=750$

Section 4価格規制と死荷重

政府が価格に介入するとき、市場にどのような歪みが生じるか？

上限価格（price ceiling）

価格が上限を超えて取引してはいけない規制。

• 上限価格 ＜ 均衡価格 → 超過需要が発生

• 上限価格 ≥ 均衡価格 → 均衡価格がそのまま実現

下限価格（price floor）

価格が下限を下回って取引してはいけない規制。

• 下限価格 ＞ 均衡価格 → 超過供給が発生

• 下限価格 ≤ 均衡価格 → 均衡価格がそのまま実現

定義：死荷重（deadweight loss）

市場の歪みによって生まれる社会的余剰の減少。死重損失とも呼ばれる。
