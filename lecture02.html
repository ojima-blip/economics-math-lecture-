キー概念

関数と写像

定義 — 写像 (mapping)

片方を１つ決めると、もう片方が１つに決まるルールのことである。

定義 — 関数 (function)

写像を数字だけに絞ったもの。片方の数字を１つ決めると、もう片方の数字が１つに決まるルールのことである。

説明変数と被説明変数

用語別名役割
説明変数独立変数決める数字（入力）
被説明変数従属変数決まる数字（出力）

１次関数と線形性

定義 — １次関数

説明変数の次数が１である関数。グラフで書くと直線で表すことができる。

定義 — 比例関数（線形関数）

１次関数のうち、定数項（切片）が０である関数。

ステップ解説

例題 1.2 ― 自動車工場の生産台数

「前へ」「次へ」ボタンで1ステップずつ確認できます。

1 / 5
1

問題を読む

工場Aでは、１日あたり100台の自動車が生産されている。

  • （１）時間を t（日）とし、工場A の t 日間の累計生産台数を X(t) で表せ。
  • （２）工場B（１日200台）も含めた総累計生産台数を Y(t) で表せ。

まず「何が説明変数で、何が被説明変数か」を確認しましょう。

2

変数を整理する

変数意味種類
t経過日数（日）説明変数（独立変数）
X(t)工場Aの累計生産台数被説明変数（従属変数）
Y(t)工場A＋Bの累計生産台数被説明変数（従属変数）

t が決まると X(t) が決まる → 関数の関係です。

3

（１）工場Aの関数 X(t) を求める

１日に100台生産する → t 日間では…

// 1日あたりの生産台数 × 日数
X(t) = 100 × t
X(t) = 100t

// 確認: t=0 → X(0)=0, t=1 → X(1)=100, t=10 → X(10)=1000

これは比例関数（切片=0の１次関数）です。

4

（２）工場A＋Bの関数 Y(t) を求める

線形性（加法性）を使います：

// 工場Bの関数
B(t) = 200t

// 線形性（加法性）: f(X+Y) = f(X) + f(Y)
Y(t) = X(t) + B(t)
Y(t) = 100t + 200t
Y(t) = 300t
5

まとめと確認

関数式t=10日後
X(t)（工場A）100t1,000台
B(t)（工場B）200t2,000台
Y(t)（合計）300t3,000台
ポイント

比例関数は線形性を持つので、Y(t) = X(t) + B(t) という足し算が成立します。これが加法性です。
